# StudySync

StudySync is a browser-based study session tracker designed to help students manage their study time, track progress, and stay consistent with their learning goals.

## ✨ Features

- **Dashboard** — View your current study streak, today's hours, total sessions logged, and daily goal progress
- **Log a Session** — Record study sessions by subject, duration, date, and optional notes with full form validation
- **Progress Charts** — Weekly bar chart and subject doughnut chart powered by Chart.js, updating in real time
- **Goals** — Set a daily study hour target that reflects on the dashboard progress bar
- **Persistent Storage** — All data saved to localStorage so nothing is lost on page reload

## 🛠️ Built With

- HTML5
- CSS3
- Vanilla JavaScript
- [Chart.js](https://www.chartjs.org/)
